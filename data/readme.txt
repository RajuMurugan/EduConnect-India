How to use:
1) Open index.html locally or deploy the folder to GitHub Pages.
2) Add more data into the files under /data.
3) For Engineering ordering: NIRF > IIT > NIT > IIIT > CFTI > Deemed > State-University (configured in engineering.html).

Notes:
- NIRF list currently includes Top 30 to keep file size readable here. You can paste the remaining 31–100 in data/nirfEngineering.js.
